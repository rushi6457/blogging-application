# Full Stack blogging application

## Tech Stacks:
<ul>
    <li>Express Js</li>
    <li>Node Js</li>
    <li>MongoDb</li>
    <li>React Js</li>
    <li>Chakra-ui</li>
    <li>CSS</li>
    <li>Json web token</li>
    <li>Swiper</li>
    <li>Chart Js</li>
    <li>Swiper</li>
</ul>

## Features of the application
<ul>
<li>Token based authentication</li>
<li>Multi user application</li>
<li>Dynamic routing</li>
<li>CRUD</li>
</ul>
